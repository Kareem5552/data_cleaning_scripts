# Data Cleaning Scripts

This repository contains Python scripts for basic data cleaning and preprocessing. 
It includes tasks like removing numbers, stripping spaces, capitalizing text, and other common string transformations.

## Features
- Remove numbers from strings
- Strip leading/trailing spaces
- Capitalize words
- Additional utility functions for list processing
